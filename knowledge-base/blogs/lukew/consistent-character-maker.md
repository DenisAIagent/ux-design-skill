# Consistent Character Maker Update

**Date:** March 17, 2026
**Author:** Luke Wroblewski
**Source:** https://lukew.com/ff/entry.asp?2146

Following initial success with the LukeW Character Maker (4,500+ characters generated), technical improvements for consistency.

## New Image Model
Google's Nano Banana 2 delivers more dynamic and three-dimensional results. Challenge: model stopped returning images matching reference dimensions.

## Prompt Rewriter Refinement
Key adjustments:
- **Faceless design reinforcement:** Multiple iterations to prevent facial features
- **Color accuracy:** Shifted to explicit HEX and RGB codes
- **Content moderation:** Progressively blocked inappropriate categories

"Start broad, then iteratively tighten consistency while expanding content moderation guardrails." Current: ~98% consistency rate.
