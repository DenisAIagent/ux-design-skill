# Design Patterns For AI Interfaces

**Date:** July 14, 2025
**Author:** Vitaly Friedman
**Source:** https://www.smashingmagazine.com/2025/07/design-patterns-ai-interfaces/

## Five Core Areas for AI Design

### 1. Input UX: Expressing Intent
Minimize typing through AI-generated pre-prompts, visual node editors, and canvas-based inputs.

### 2. Output UX: Displaying Outcomes
Visualize contextually through style lenses, maps, data tables — not plain text.

### 3. Refinement UX: Tweaking Output
Sliders, knobs, buttons for fine-tuning. Highlight sections for targeted prompts.

### 4. AI Actions: Tasks to Complete
Delegate work (scheduling, research, filtering) to AI agents with suggested actions.

### 5. AI Integration: Where Work Happens
Integrate into existing workflows (Slack, Jira, GitHub) — transparent co-pilots.

## Core Recommendation
Prioritize being "AI-second" rather than AI-first, enhancing existing mental models.
