# Inactive GUI Controls: Show, Disable, or Hide?

**Date:** 2025-11-13T12:01:07.310Z
**Author:** Jakob Nielsen
**Source:** https://www.uxtigers.com/post/inactive-buttons

---

Three design patterns exist for inactive UI commands: showing them as normal commands with an error message if clicked, a visibly disabled state, or completely hiding unavailable options. Best practice is to display disabled features in muted colors, accompanied by explanations of why they are currently unavailable.
