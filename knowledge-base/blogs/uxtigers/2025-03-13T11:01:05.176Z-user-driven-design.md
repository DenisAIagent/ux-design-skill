# Does User-Driven Design Still Need User-Centered Design?

**Date:** 2025-03-13T11:01:05.176Z
**Author:** Jakob Nielsen
**Source:** https://www.uxtigers.com/post/user-driven-design

---

UX shifts radically as "vibe coding" places users in control of digital functionality. AI enables rapid, inexpensive development, empowering domain specialists to design software independently. UX professionals must pivot to scaling effective user innovations, guiding emergent patterns, and embracing a strategic role instead of a design role.
