# A Concrete Definition of an AI Agent

**Date:** April 3, 2026
**Author:** Caleb Sponheim
**Source:** https://www.nngroup.com/articles/definition-ai-agent/

"An AI agent pursues a goal by iteratively taking actions, evaluating progress, and deciding its own next steps."

## Core Components

- **Pursues a Goal:** Specific or open-ended objectives
- **Iteratively:** Builds upon previous work
- **Taking Actions:** Searches web, writes code, edits documents
- **Evaluating Progress:** Reviews completed and remaining work
- **Deciding Its Own Next Steps:** Self-direction distinguishes agents

## Two-Axis Framework

- **Capability to Act:** None to unconstrained
- **Self-Direction:** Predetermined vs. self-determined

## Useful Agent Criteria

- Reliable goal comprehension
- Failure adaptation
- Minimal supervision
